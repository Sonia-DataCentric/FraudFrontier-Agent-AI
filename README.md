**🛡️ FraudFrontier-Agent-AI (Phase 1)**
An Autonomous AI Agent for Financial Data Ingestion & Compliance-Ready Preprocessing.

**📌 Project Vision**
Traditional fraud detection pipelines fail because of "dirty" data or missed manual cues. FraudFrontier-Agent-AI transitions from manual "gut-feeling" verification to an autonomous system.

Phase 1 uses Agentic AI (CrewAI) to ingest messy financial logs, identify PII (Personally Identifiable Information), and auto-clean data specifically for AML/KYC Risk Scoring.

**🚀 Key Features**
Intelligent Column Mapping: Detects banking-specific headers (e.g., PAN, Transaction Amount, UBO IDs) even in unlabelled CSVs.

Compliance-First Cleaning: Automatically handles missing values using RBI-standard logic (e.g., flagging transactions near the ₹50,000 threshold).

Velocity Feature Engineering: Prepares timestamps for frequency-based fraud detections
